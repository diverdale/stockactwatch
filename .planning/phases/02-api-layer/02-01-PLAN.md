```python
METHODOLOGY_LABEL_V1 = "v1 — estimated gain/loss vs. entry price (midpoint of disclosed range)"

    full_name: str

class LeaderboardResponse(BaseModel):
    entries: list[ReturnLeaderboardEntry]
    total: int
    cached: bool

    total: int
    cached: bool
    filters_applied: dict
```

**app/api/leaderboard.py** — create with:

1. Enum definitions at module top:
```python
from enum import Enum

class Chamber(str, Enum):
    house = "House"
    senate = "Senate"

class Party(str, Enum):
    democrat = "Democrat"
    republican = "Republican"
    independent = "Independent"

class TimePeriod(str, Enum):
    ytd = "ytd"
    one_year = "1y"
    all_time = "all"
```

2. Two standalone async query functions (not inside the route handler — needed for unit testing):
   - `query_returns_leaderboard(db: AsyncSession, limit: int = 20) -> list[Row]`
     Uses SQLAlchemy 2.0 `select(Politician.id, Politician.full_name, func.avg(...).label("avg_return_pct"), func.min(...).label("return_low"), func.max(...).label("return_high"), func.count(ComputedReturn.id).label("trade_count"), func.max(ComputedReturn.methodology_ver).label("methodology_ver"))` JOINed on `ComputedReturn.politician_id == Politician.id`, with `.where(ComputedReturn.return_pct.is_not(None))`, grouped by `Politician.id, Politician.full_name`, ordered by `func.avg(ComputedReturn.return_pct).desc()`, limited.
     Returns `result.all()` — NOT `result.scalars().all()` (scalars() breaks on GROUP BY tuples).
   - `query_volume_leaderboard(db: AsyncSession, chamber: str | None, party: str | None, period: str, limit: int = 20) -> list[Row]`
     Builds dynamic WHERE with only non-None filters. Uses `_period_to_cutoff(period)` helper that maps "ytd" → `date(today.year,1,1)`, "1y" → `today - timedelta(365)`, "all" → None. Selects `Politician.id, Politician.full_name, Politician.chamber, Politician.party, func.count(Trade.id).label("trade_count")`, JOINed on `Trade.politician_id == Politician.id`, grouped by politician fields, ordered by trade_count desc.

3. Cache key helpers (for use in Plan 02-02):
   - `returns_cache_key(limit: int) -> str` — returns `f"leaderboard:returns:lim={limit}"`
   - `volume_cache_key(chamber, party, period, limit) -> str` — returns `f"leaderboard:volume:ch={chamber or 'any'}:p={party or 'any'}:t={period}:lim={limit}"`

4. `router = APIRouter(prefix="/leaderboard", tags=["leaderboard"])`

5. `GET /leaderboard/returns` route:
   - Parameters: `limit: Annotated[int, Query(ge=1, le=100)] = 20`
   - Dependency: `db: AsyncSession = Depends(get_db)`
   - Calls `query_returns_leaderboard(db, limit)`
   - Maps rows to `ReturnLeaderboardEntry` — `politician_id=str(row.id)`, `methodology_label=METHODOLOGY_LABEL_V1`, `disclaimer=True`
   - Returns `LeaderboardResponse(entries=entries, total=len(entries), cached=False)`
   - `response_model=LeaderboardResponse`

6. `GET /leaderboard/volume` route:
   - Parameters: `chamber: Annotated[Chamber | None, Query()] = None`, `party: Annotated[Party | None, Query()] = None`, `period: Annotated[TimePeriod, Query()] = TimePeriod.all_time`, `limit: Annotated[int, Query(ge=1, le=100)] = 20`
   - Dependency: `db: AsyncSession = Depends(get_db)`
   - Calls `query_volume_leaderboard(db, chamber.value if chamber else None, party.value if party else None, period.value, limit)`
   - Maps rows to `VolumeLeaderboardEntry`
   - Returns `VolumeLeaderboardResponse(entries=entries, total=len(entries), cached=False, filters_applied={"chamber": chamber, "party": party, "period": period, "limit": limit})`
   - `response_model=VolumeLeaderboardResponse`

```python
from fastapi import Request
from fastapi.encoders import jsonable_encoder
from fastapi.exceptions import RequestValidationError

@app.exception_handler(RequestValidationError)
async def validation_exception_handler(request: Request, exc: RequestValidationError):

        status_code=422,
        content={"detail": exc.errors(), "body": jsonable_encoder(exc.body)},
```
